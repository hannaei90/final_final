Assuming that python and an IDE already is installed on the computer.
Download the zip file and open controller.py in a virtual environment.
Download flask and sqlite3 in the terminal (pip install flask/sqlite3/pysqlite3).
Run controller.py and open the local server.
On the website, write a location in "Plats", the available locations are in our database:
    places2.db [ica, coop, hemköp, synoptik, synsam, postnord, posten, polisen, frisören, migrationsverket, skatteverket].
